About this Project

Built with Python, Scikit-learn & Flask

The classic Iris dataset contains 150 flower records across three species — Iris-setosa, Iris-versicolor, and Iris-virginica — each described by four measurements: sepal length, sepal width, petal length, and petal width. The goal is to accurately classify which species a flower belongs to based purely on these measurements.

"Can a machine learn to distinguish flowers just by their measurements — the way a botanist would with a ruler?"

🔍 Exploratory Data Analysis (EDA)

The project began with thorough EDA. After loading the dataset, we discovered 3 duplicate rows which were removed, bringing the working dataset to 147 clean records. No missing values were found across any of the 5 columns.
